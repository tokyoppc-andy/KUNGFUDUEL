# **App Name**: Kung Fu Duel Sandbox

## Core Features:

- Character State Management: Manage HP, Energy, ComboHits, and EvadeStreak for player and AI.
- Move Clash Simulation: Simulate player and AI moves in a 'Move Clash' phase to determine attack priority.
- Directional Combat: Simulate directional attacks and defenses, calculating hits and misses.
- Elemental & Special Abilities: Implement unique abilities based on character elements (Fire, Earth, etc.) and special moves.
- Game Loop Progression: Advance the game through rounds, managing turn order and special condition triggers.
- Combat Outcome Display: Display the results of each round, including character stats and combat events.
- Post-Game Analysis & Visualization: Calculate and display post-game statistics and 'radar chart' visualizations.

## Style Guidelines:

- Primary color: Deep fiery red (#B80F0A) representing passion and combat intensity.
- Background color: Dark charcoal grey (#282828) providing a stark contrast for vibrant action.
- Accent color: Bright, energetic orange (#FF8C00) for energy indicators and important call-to-actions.
- Font pairing: 'Space Grotesk' for dynamic headlines and 'Inter' for readable body text.
- Use bold, striking icons that evoke traditional East Asian art styles for abilities and elements.
- Clean, modern layout with distinct sections for player stats, AI stats, and combat logs. Emphasize asymmetry for a dynamic feel.
- Fast, impactful animations for attacks, defenses, and special ability activations. Use subtle transitions for UI element updates.
